Quext

Quext helps apartment owners and operators create better communities through technology. Its products are designed to work seamlessly in multifamily environments and enable users to build the kind of community where people love to live and work.

The main clients of the company are property owners, operators, and residents. I worked on the analytical part for operators.

Problem

To effectively manage a large number of properties, managers need access to different information based on their area of responsibility. This requires a customized approach, including dashboards or tabular reports.

Task

Conveniently display the necessary data for different user groups

Metrics for Evaluating Work

  • Interaction rate
  • Interface predictability
  • Number of transitions

Solution

Interviews with various user groups revealed that even when using the same data, they obtain different information and have different goals. Many people require different sets of reports for their work, which can only be achieved through workspace customization or by predefining the structure.

Approach

We created a dashboard with customizable widgets that can be rearranged based on user preference. We also added the ability for admins to save custom layouts and share them with team members. This streamlined the process of accessing and analyzing data, leading to a higher interaction rate and improved interface predictability. Additionally, we reduced the number of transitions required to access different sections of the dashboard, making it more efficient for users.

Universality

Each component has to be universal so that it can be used by other vendors and engineering teams. To achieve this, we use design tokens to describe each design solution and sync every design entity with its corresponding front-end solution.

Iteration

We conducted user testing on the dashboard and received valuable feedback. Based on this feedback, we made adjustments to the layout and design of the widgets to improve user experience. We also added a feature that allows users to quickly filter and sort data within the dashboard. These changes led to an increase in the interaction rate and a more efficient workflow for users.

Final Product

The final product is a customizable dashboard that allows property managers to access and analyze data in a streamlined and efficient way. The use of design tokens ensures that each component is universal and can be used by other vendors and engineering teams. The dashboard has been well-received by users and has led to an increase in productivity and overall satisfaction.
